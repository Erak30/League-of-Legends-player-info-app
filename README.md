# League-of-Legends-player-info-app

A League of Legends match viewer that retrieves a player's latest match data using the Riot Games API. By entering a Riot ID and region, the app displays the champion played, match result, rank, and the items used in the most recent game.

# Setup

To test this project  you need to put in your own developer API key you can generate from the Riot Developer Portal. In the models folder there is a file called Key.cs, within this file place your key, you should now be able to run the app.
